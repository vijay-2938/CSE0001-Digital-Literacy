For this task, I shifted my focus toward how digital platforms can simplify two very different types of work: logical problem-solving and data organization. I started by heading over to HackerRank to tackle a coding challenge. I chose to solve the "Solve Me First" problem using Python (you can change this to C++ or Java if needed). While the logic was straightforward, it was a great exercise in understanding how automated platforms validate code against various test cases. It really highlighted the importance of precision in programming—even a small syntax error can stop a project in its tracks.

Moving from code to data collection, I used Google Forms to design a quick 5-question quiz on Digital Literacy. I wanted to see how much people actually know about things like two-factor authentication and identifying phishing attempts. The most useful part of this was linking the form to a Google Sheet. Watching the responses populate a spreadsheet automatically made me realize how much time this saves compared to manual data entry.

This task taught me that being a "Digital Ambassador" isn't just about knowing how to code; it’s about knowing which tools to use to get a job done efficiently. Whether it’s solving a math problem through code or organizing survey results, these platforms take the "busy work" out of the process, allowing for much faster decision-making.
